Integración Continua con GitHub Actions

Pipeline de CI configurado con GitHub Actions que ejecuta automáticamente en cada push. Build en entorno containerizado con Docker. Ejecución paralela de suites de pruebas. Generación de artifacts para debugging. Notificaciones en Slack de resultados. 

# .github/workflows/ci-cd.yml
        name: CI/CD Pipeline
        on: [push, pull_request]

        jobs:
          quality-checks:
            runs-on: ubuntu-latest
            steps:
              - uses: actions/checkout@v3
              - uses: actions/setup-node@v3
                with: { node-version: '18' }
              
              - name: Install dependencies
                run: npm ci
              
              - name: Run linting
                run: npm run lint
              
              - name: Execute unit tests
                run: npm run test:unit -- --coverage
              
              - name: Run E2E tests
                uses: cypress-io/github-action@v5
                with:
                  browser: chrome
                  headless: true
              
              - name: Upload test results
                uses: actions/upload-artifact@v3
                if: failure()
                with:
                  name: cypress-screenshots
                  path: cypress/screenshots

# Workflow de despliegue condicional
                deploy-to-production:
                  name: Deploy to Production
                  needs: [quality-checks, e2e-tests]
                  if: github.ref == 'refs/heads/main' && needs.quality-checks.result == 'success'
                  runs-on: ubuntu-latest
                  environment: production
                  steps:
                    - name: Deploy to Vercel
                      uses: amondnet/vercel-action@v20
                      with:
                        vercel-token: ${{ secrets.VERCEL_TOKEN }}
                        vercel-org-id: ${{ secrets.ORG_ID }}
                        vercel-project-id: ${{ secrets.PROJECT_ID }}
                        vercel-args: '--prod'
                    
                    - name: Run smoke tests
                      run: npm run test:smoke
                    
                    - name: Notify deployment
                      uses: 8398a7/action-slack@v3
                      with:
                        status: ${{ job.status }}
                        text: 'Deployment completado exitosamente'

DEF-042: Error en cálculo de descuento

Alta Cerrado
Resumen:

Descuento del 20% no se aplica correctamente cuando hay múltiples productos en el carrito

  1. Agregar 3 productos al carrito (total: $300)

Descuento del 20% sobre el total ($60) aplicado correctamente
